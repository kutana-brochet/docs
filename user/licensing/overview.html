Licensing

If there is a valid Licence then the application will run in full mode. If there is no valid licence then the application will run in a limited mode.

Activating a Licence

Open the About Dialog from the Ribbon and click 'Activate'.

Deactivating a Licence

Open the About Dialog from the Ribbon and click 'Deactivate'.
