Valid Licence - No Internet Connection

sequenceDiagram
autonumber
  participant A as Add-in
  participant L as Licensing Repository
  participant LF as Licence Config
  box Licensing Nuget
  participant IV as IsValid()
  participant ST as Status()
  participant RLLF as ReadLicenceLeaseFile()
  participant S as Licensing Service
  end
    A->>L: GetLicence()
    L->>LF: Validate
    LF->>L: Valid
    L->>IV: Licensing.IsValid()
    IV->>RLLF: ReadLicenseLeaseFile()
    alt Lease Exists and Active
        RLLF->>IV: No Exception
        IV->>L: return True
        L->>RLLF: Licensing.ReadLicenseLeaseFile()
        RLLF->>L: Licence Details
        L->>L: Extract Entitlements from Licence Details
        L->>A: Active Licence
        Note over L,A: Status: Active
    else Lease does not Exist or Expired
        RLLF->>S: Ping Server
        break when the server is unreachable
        S->>RLLF: Exception
        RLLF->>IV: Exception
        IV->>L: Return False
        L->>ST: GetStatus()
           Note over L,ST: We don't know why it isn't valid so need to check
        ST->>RLLF: ReadLicenseLeaseFile()
        RLLF->>S: Ping Server
        S->>RLLF: Exception
        RLLF->>ST: Exception
        ST->>L: Exception
        L->>A: Invalid Licence
        Note over L,A: Status: Server Unreachable
        end
    end

        L->>A: Active Licence
        Note over L,A: Status: Active
    else Lease does not Exist or Expired
        RLLF->>S: Ping Server
      
        S->>RLLF: OK
         RLLF->>S: Validate Licence Key
       S->>RLLF: return Licence Information 
alt Licence Active

 RLLF->>S: Checkout Licence
        S->>RLLF: return Licence Details
        RLLF->>RLLF: Write Lease File
        RLLF->>IV: No Exception
        IV->>L: return True
                L->>RLLF: Licensing.ReadLicenseLeaseFile()
        RLLF->>L: Licence Details
        L->>L: Extract Entitlements from Licence Details
        L->>A: Active Licence
        Note over L,A: Status: Active
        else Licence Not Active
          S->>RLLF: Exception
          RLLF->>IV: Exception
          IV->>L: Return False

           Note over L,ST: We don't know why it isn't valid so need to check

        RLLF->>S: Ping Server
        S->>RLLF: Exception
        RLLF->>ST: Exception
        ST->>L: Exception
        L->>L: Extract Status Reason from Exception
        L->>A: Invalid Licence

Invalid Licence

sequenceDiagram
autonumber
  participant A as Add-in
  participant L as Licensing Repository
  participant LF as Licence Config
  box Licensing Nuget
  participant IV as IsValid()
  participant S as Licensing Service
  end
  
  
    A->>L: GetLicence()
    L->>LF: Validate
    alt  LicenceId empty or invalid GUID
        LF->>L: Invalid
        L->>A: Invalid Licence
        Note over L,A: Status: Licence File missing or invalid
    else Activation Key empty
            LF->>L: Invalid
        L->>A: Invalid Licence
        Note over L,A: Status: Licence File missing or invalid
        else Activation Key is invalid
                LF->>L: Valid
                L->>IV: Licensing.IsValid()
                IV->>S: Ping Server
                S->>L: ApiException
         Note over S,L: Status Code 401 Unauthorized
                L->>A: Invalid Licence
                       Note over L,A: Status: Licence File missing or invalid
  else not Active
        LF->>L: Valid
        L->>IV: Licensing.Status()
        IV->>S: Validate Licence Key
        S->>L: KeygenException
                Note over S,L: Contains <Status Reason>
        L->>A: Invalid Licence
        Note over L,A: Status: <Status Reason>
    else Any other validation failure
        L->>A: Invalid Licence
        Note over L,A: Status: Undefined
    end
