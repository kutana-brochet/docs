NetDocuments Event handling

Calling the NetDocuments saveAs dialog via their ComAddin.Object means that it runs synchronously so need something like System.Windows.Forms.Application.DoEvents. Calling it via the Word Application.CommandBars.ExecuteMso "FileSaveAs" it runs asynchronously.

There are only 5 events

Event Description
NdAfterSaveEvent Occurs after ndOffice Save.
NdBeforeBrowseEvent Occurs before Save As dialog appears

Remarks
Using this event you can possibly change default profile attributes for new documents to show on the Save As dialog.
NdBeforeSaveEvent Occurs before ndOffice Save.
NdDocumentReadyEvent Occurs when user document was opened and ready for processing
NdEditProfileEvent Occurs before ND document content saved inside echo location.

Remarks Using this event you can change profile attributes after dialogs and update document
NdEditProfileResultEvent Occurs when user edits profile attributes in NdSetProfile event. (0 for OK 1 for cancel)
NdSaveDialogResultEvent Occurs when user closes save dialog. (0 for OK 1 for cancel)
NdUnsavedChangesResultEvent Occurs when prompt about unsaved changes is closed. (0 Cancel, 1 don't save)
